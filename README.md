# BagNest

BagNest is a full-stack e-commerce web application for shopping different types of bags. Users can create an account, log in securely, browse products, and manage their shopping experience.

## Features
- User Registration and Login
- JWT Authentication & Authorization
- Password Hashing with bcrypt
- Admin can create a product
- Product Listing
- Adding product in cart
- Order Listing
- User profile
- Track products
- MongoDB Database Integration
- Dynamic UI using EJS
- Responsive Design with Tailwind CSS

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- Tailwind CSS
- JWT
- bcrypt
- Javascript
